# GAINZ — AI Fitness Coach

AI-powered fitness app. Add to iPhone home screen via Safari.

## Setup for GitHub Pages

1. Push this folder to a GitHub repo
1. Go to **Settings → Pages → Source: main branch / root**
1. Open `https://yourusername.github.io/your-repo-name/` in Safari
1. Tap **Share → Add to Home Screen**

## Features

- 🤖 AI Coach (powered by Claude)
- 📋 Daily Routines with points
- ⚔️ Daily Side Quests
- 🏋️ Workout tracking + progressive overload
- ⚡ AI-generated workout plans by equipment
- 🍽️ Nutrition targets + recipes
- 🏆 Points, streaks & leaderboard
